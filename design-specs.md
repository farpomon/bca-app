# Professional UI Design Specifications (From Canva)

## Color Palette

### Primary Colors
- **Dark Navy Background**: `#1e2a3a` (main background)
- **Slate Blue**: `#3d4f5f` (secondary background, cards)
- **Professional Blue**: `#2196F3` (accents, CTAs)
- **Light Blue**: `#64B5F6` (hover states, highlights)

### Text Colors
- **Primary Text (White)**: `#FFFFFF`
- **Secondary Text (Light Gray)**: `#B0BEC5`
- **Muted Text**: `#78909C`

### Accent Colors
- **Success Green**: `#4CAF50`
- **Warning Orange**: `#FF9800`
- **Error Red**: `#F44336`

## Typography

### Font Family
- **Primary**: 'Inter', 'Segoe UI', sans-serif
- **Headings**: 'Inter', weight 600-700
- **Body**: 'Inter', weight 400-500

### Font Sizes
- **Hero Title**: 56px / 3.5rem (Building Condition Assessment)
- **Section Title**: 32px / 2rem (Professional Services)
- **Card Title**: 24px / 1.5rem
- **Body Large**: 20px / 1.25rem
- **Body**: 16px / 1rem
- **Small**: 14px / 0.875rem
- **Tiny**: 12px / 0.75rem

## Layout & Spacing

### Container
- **Max Width**: 1400px
- **Padding**: 80px horizontal, 60px vertical

### Card Styling
- **Background**: `#3d4f5f` with subtle gradient
- **Border Radius**: 16px
- **Padding**: 32px
- **Shadow**: 0 8px 32px rgba(0, 0, 0, 0.3)
- **Hover Effect**: Lift with increased shadow

### Spacing Scale
- **xs**: 4px
- **sm**: 8px
- **md**: 16px
- **lg**: 24px
- **xl**: 32px
- **2xl**: 48px
- **3xl**: 64px

## Component Styles

### Buttons
- **Primary**: Blue gradient background, white text, 12px border radius
- **Height**: 48px (large), 40px (medium), 32px (small)
- **Padding**: 24px horizontal
- **Font Weight**: 600
- **Hover**: Brighten 10%, lift 2px

### Cards
- **Background**: Dark slate with subtle gradient
- **Border**: None or 1px solid rgba(255,255,255,0.1)
- **Hover**: Transform translateY(-4px), shadow increase

### Stats/Metrics Cards
- **Number Size**: 48px, weight 700
- **Label Size**: 14px, weight 500, uppercase
- **Icon Size**: 32px
- **Spacing**: 16px between elements

## Professional Design Principles

1. **Dark Professional Theme**: Navy/slate color scheme
2. **High Contrast**: White text on dark backgrounds
3. **Generous Spacing**: Breathing room between elements
4. **Subtle Gradients**: Depth without distraction
5. **Soft Shadows**: Elevation and hierarchy
6. **Clean Typography**: Sans-serif, clear hierarchy
7. **Professional Imagery**: High-quality building photos
8. **Consistent Rounding**: 12-16px border radius throughout
